nmb -- NetBIOS over TCP/IP operations

This module provides basic NetBIOS over IP functionalities.

Exported Classes

NetBIOSError (Exception)
This exception is raised when an error has occured when performing a NetBIOS operation.
NetBIOSTimeout (Exception)
This exception is raised when a timeout has occured when waiting for reply to a NetBIOS operation.
NBHostEntry
Instances of this class contain information of the resolved host for a NetBIOS name.
NBNodeEntry
Instances of this class contain information about the nodes.
NetBIOS
A wrapper class which contains the required NetBIOS over IP operations for pysmb.
NetBIOSSession
Represents a NetBIOS over TCP/IP session. You should not create an instance of this class directly. This class is used primarily by smb module.

Exported Functions

strerror(errclass, errcode) -> string, string
Returns the corresponding messages for errclass and errcode.
